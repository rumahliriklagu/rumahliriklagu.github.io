27. Perbandingan uang Afgan dan Sulthanul yaitu 8 : 3. Jika selisih uang mereka Rp 25.000,00, maka uang Afgan adalah​


Jawaban:

Jumlah uang Afgan adalah Rp. 40.000,00

Penjelasan dengan langkah-langkah:

misal

uang afgan = A

uanv sulthanul = B

A : B = 8 : 3

A - B = 25.000

A = 25000 + B

(25.000 + B) : B = 8 : 3

3(25.000 + B) = 8B

75.000 + 3B = 8B

75.000 = 5B

B = 15.000

A = 25.000 + 15.000

A = 40.000

Jadi, jumlah uang afgan adalah Rp. 40.000,00
