19. If i can....the target,my boss....give me reward


Conditional sentence :

✓ tells about the condition that will be realized ( possible condition ) and only dreaming ( intention )

✓ there are 3 types

✓ Consists -If clause and result clause.

✓ Uses Comma if the -If clause is started at the beginning sentence.

1. Conditional type 1

✓ states the true in the present/future

✓ states the fact

✓ the speaker will actualize his/her dream

✓ the speaker tells about the phenomenon and non phenomenon

✓ uses simple present tense and simple future

Pattern :

If + S + verb1 + O....., S+will+verb1+O

Example :

1. If I can reach the target, my boss will give me reward.

Condition : My boss probably gives me reward

2. If it rains, the roads get wet.

Condition : this is the fact. The natural phenomenon. This is habitual activity.

2. Conditional sentence type 2

✓ States the untrue in the present / future

✓ Contrary to the fact

✓ The speaker has intention but he/she can't afford his/her dream to be come true in the present/future

✓uses simple past and past future

✓uses Tobe ( were) for all subjects ( either singular or plural )

Pattern:

If+S+verb2+O....S+Would+verb1+O

Example :

If I were a president, I would settle down the economic issues.

Condition : I'm not a president, so I can't settle down the economic issues.

3. Conditional sentence type 3

✓ States the untrue in the past

✓ Contrary to the fact

✓ uses past perfect and past future perfect

Pattern :

If+S+Had+V3+O....,S+Would+have+V3

Example :

If they had studied, they would have passed the exam.

Condition : they didn't study. Therefore, they failed the exam.

Penjelasan:

perbedaan efek fisiologi dan efek mengkonsumsi depresan adalah

SEMOGA MEMBANTU

➡️ 3 efek fisiologi dari depresan

1) menyebabkan kecemasan

➡️ 3 efek fisik konsumsi depresan

➡️ Pembahasan

Depresan adalah obat yang di pergunakan untuk mengobati atau menangani seseorang yang terkena depresi. Cara kerja obat ini yaitu menyeimbangkan senyawa kimia di dalam otak kita sehingga bisa membantu meredakan rasa sakit yang kita alami.

Obat hanya memberikan kesenangan sementara badan menjadi lemas dan mudah mengantuk atau bahkan tak sadarkan diri. Maka kita harus benar benar menjauhinya. Tidak hanya pada fisik kita tetapi psikologis kita juga akan terganggu apabila terus menerus meminumnya.
