7. 1. Mana kalimat perintah untuk objek “Vous “ ? A. parles !B. écoutez !C. danse !D. chantons !E. allez2. Mana kalimat perintah untuk objek “Nous “ ? A. parles !B. écoutez !C. danse !D. chantons !E. allez3. Mana kalimat perintah untuk objek “vous “ ? A. finis !B. finir !C. finissons !D. finissez !E. finit4. Mana kalimat perintah untuk objek “Tu “ ? A. choisis !B. choisit !C. choisissons !D. finissez !E. finissent5. Mana kalimat perintah untuk objek “Nous “ ? A. brunit !B. brunis !C. brunissons !D. brunir !E. brunissez !6. Mana kalimat perintah untuk objek “Tu “ ? A. prend !B. prends !C. prenez !D. prenons !E. prennent !7. Mana kalimat perintah untuk objek “Vous “ ? A. bois !B. buvez !C. boit !D. boire !E. boi !8. Mana kalimat perintah untuk objek “Nous “ ? A. va !B. allez !C. écoute !D. chantez !E. allons !


L'impératif

1. Écoutez !

2. Chantons !

3. Finissez !

4. Choisis !

5. Brunissons !

6. Prends !

7. Buvez !

8. Allons !

ーーーーーーーーーーーー

Explication

L'impératif atau modus imperatif adalah bentuk kata kerja yang digunakan untuk memberi perintah.

E.g.

Ouvrir / membuka ⇨ Ouvrez ! / bukalah!Aller / pergi ⇨ Allons-y ! / Ayo pergi!

Konjugasi kata kerja dalam modus imperatif ada tiga, yaitu bentuk tu, nous, dan vous.

Secara umum, ketiga bentuk ini sama dengan konjugasi présent untuk subyek tu, nous, dan vous.

Par exemple ...

Présent untuk kata kerja partir:

je pars tu pars il/elle/on part nous partons vous partez ils/elles partent

Impératif untuk kata kerja partir:

(tu) pars ! (nous) partons ! (vous) partez !

Untuk kata kerja berakhiran -er, bentuk imperatifnya menjadi berakhiran -e, -ons, dan -ez.

Contoh:

Regarder

(tu) regarde ! (nous) regardons ! (vous) regardez !

Kalimat imperatif selalu diakhiri le point d'exclamation/tanda seru (!).

15. mohon bantuanyaKonjugasikan kata kerja di bawah ini!ParlerFinir DormirFaireprendremettrePouvoirSavoirBoireLireCara mengkonjugasikan, kerjakan seperti contoh dibawah ini:Aller = pergiJe vaisTu vasIl/elle vaNous allonsVous allezIls/elles vont​


Mengkonjugasikan kata kerja

( le présent de l'indicatif )

1. Parler

Je => parleTu => parlesIl/elle/on => parleNous => parlonsVous => parlezIls/elles => parlent

2. Finir

Je => finisTu => finisIl/elle/on => finitNous => finissonsVous => finissezIls/elles => finissent

3. Dormir

Je => dorsTu => dorsIl/elle/on => dortNous => dormonsVous => dormezIls/elles => dorment

4. Faire

Je => faisTu => faisIl/elle/on => faitNous => faisonsVous => faitesIls/elles => font

5. Prendre

Je => prendsTu => prendsIl/elle/on => prendNous => prenonsVous => prenezIls/elles => prennent

6. Mettre

Je => metsTu => metsIl/elle/on => metNous => mettonsVous => mettezIls/elles => mettent

7. Pouvoir

Je => peux/puisTu => peuxIl/elle/on => peutNous => pouvonsVous => pouvezIls/elles => peuvent

8. Savoir

Je => saisTu => saisIl/elle/on => saitNous => savonsVous => savezIls/elles => savent

9. Boire

Je => boisTu => boisIl/elle/on => boitNous => buvonsVous => buvezIls/elles => boivent

10. Lire

Je => lisTu => lisIl/elle/on => litNous => lisonsVous => lisezIls/elles => lisent

Kata kerja dalam bahasa Prancis digolongkan menjadi 3 macam, yakni kata kerja kelompok pertama atau 1er groupe, kata kerja kelompok kedua atau 2e groupe, dan kata kerja kelompok ketiga atau 3e groupe, serta kata kerja apabila akan dipakai dalam suatu kalimat seharusnya diubah/dikonjugasikan sesuai dengan aturan-aturan yang ada.

18. futur proche 1. Le train....... (aller partir)dans 5 minutes. *a. allez partirb. vais partirc. va partird. vas partir​


Le futur proche

Le train va partir dans 5 minutes.

ーーーーーーーーーーーー

Préface

Le futur proche adalah temps yang digunakan untuk menyatakan aksi/keadaan yang akan segera terjadi.

Futur proche terdiri dari aller dalam bentuk présent + infinitif

Par exemples ...

Je vais sortir / saya akan pergiTu vas étudier / kamu akan belajar

Konjugasi aller dalam présent adalah:

je vais / saya akantu vas / kamu akanil/elle/on va / dia akannous allons / kami/kita akanvous allez / Anda/kalian akanils/elles vont / mereka akan

ーーーーーーーーーーーー

Explication

Le train = kereta itu adalah benda tunggal. Kata ganti untuk le train adalah il. Jadi, aller partir berubah menjadi va partir (opsi c).

Le futur proche

Ta mère va faire du gâteau pour ses enfants.

ーーーーーーーーーーーー

Préface

Le futur proche adalah bentuk kata kerja yang digunakan untuk menyatakan aksi/keadaan yang akan segera terjadi dalam waktu dekat.

Futur proche terdiri dari aller dalam konjugasi présent dan infinitif.

Par exemples ...

Il va nager demain / Saya akan berenang besok.Ce soir, tu vas arriver à Paris / Nanti malam, kamu akan tiba di Paris.

Konjugasi aller dalam présent adalah:

je vais / saya akantu vas / kamu akanil/elle/on va / dia akannous allons / kami/kita akanvous allez / Anda/kalian akanils/elles vont / mereka akan

ーーーーーーーーーーーー

Explication

Ta mère = ibumu = dia = elle. Jadi, aller faire berubah menjadi va faire (opsi d).

40. Comment allez-vous ?​


Comment allez-vous?

Dalam bahasa Indonesia : Bagaimana kabar anda?

Jawaban : Je vais très bien, merci. Et vous?

Terjemah jawaban : Saya sangat baik-baik saja, terima kasih. Dan anda?

Pembahasan

Comment allez-vous? merupakan frasa bahasa Prancis yang digunakan untuk menanyakan kabar atau keadaaan seseorang dalam kondisi formal. Frasa tersebut diucapkan pada orang asing, pada acara formal (resmi), pada orang yang lebih tua, atau pada orang yang sangat dimuliakan.

⇒ Comment berarti bagaimana.

⇒ Allez berasal dari kata aller berarti pergi (dalam bahasa Inggris sama seperti kata "go" dikonjugasikan dengan subjek menjadi "be going").

⇒ Vous berarti anda (bentuk formal untuk mengatakan "tu").

Ada beberapa frasa yang digunakan untuk menanyakan kabar dalam bahasa Prancis selain menggunakan "Comment allez vous" misalnya :

Frasa "Comment ça va" digunakan dalam kondisi informal. Frasa ini diucapkan hanya kepada orang-orang dekat yang sudah dikenal.Frasa "Ça va" digunakan dalam kondisi informal. Sifatnya lebih informal daripada frasa "Comment ça va".Frasa "Comment vas-tu" digunakan dalam kondisi informal, tetapi bersifat lebih sopan jika digunakan. Frasa ini diucapkan pada teman.

Untuk menjawab pertanyaan-pertanyaan tentang kabar tersebut, bisa digunakan beberapa kalimat misalnya seperti berikut ini :

⇒ Je vais très bien (saya sangat baik-baik saja).

⇒ Très bien (sangat baik).

⇒ Bien (baik).

⇒ Assez bien (cukup baik).

⇒ Je vais mal (saya tidak baik-baik saja).

⇒ Comme-ci comme-ca (biasa saja).

Tambahan :

Agar terdengar lebih ramah, maka setelah menjawab pertanyaan kabar tersebut, kita menambahkan kata "Merci" berarti "terima kasih".

Dan untuk menanyakan kembali pertanyaan pada orang yang beranya, bisa ditambahkan "Et vous?" berarti "Dan anda?" (dalam situasi formal) atau "Et toi?" berarti "Dan kamu" (dalam situasi informal).
