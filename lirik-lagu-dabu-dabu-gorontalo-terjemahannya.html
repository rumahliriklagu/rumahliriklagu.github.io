20. -kuis kecil2an part 2- passive voice dari "Mawar is eating mujair fish with dabu-dabu"


Jawaban:

ᴀᴄᴛɪᴠᴇ : ᴍᴀᴡᴀʀ ɪs ᴇᴀᴛɪɴɢ ᴍᴜᴊᴀɪʀ ғɪsʜ ᴡɪᴛʜ ᴅᴀʙᴜ-ᴅᴀʙᴜ

ᴘᴀssɪᴠᴇ : ᴍᴜᴊᴀɪʀ ғɪsʜ ᴡɪᴛʜ ᴅᴀʙᴜ-ᴅᴀʙᴜ ɪs ʙᴇɪɴɢ ᴇᴀᴛᴇɴ ʙʏ ᴍᴀᴡᴀʀ

Penjelasan:

ᴘʀᴇsᴇɴᴛ ᴄᴏɴᴛɪɴᴜᴏᴜs :

ᴀᴄᴛɪᴠᴇ : s + ɪs/ᴀᴍ/ᴀʀᴇ + ᴠᴇʀʙ-ɪɴɢ + ᴏ

ᴘᴀssɪᴠᴇ : s + ɪs/ᴀᴍ/ᴀʀᴇ + ʙᴇɪɴɢ + ᴠ3 + ʙʏ + ᴏ

Mujair fish is being eaten (by Mawar) with dabu-dabu.

EXPLICATION :

Passive Voice :

Merupakan kalimat pasif dalam bahasa Inggris. Sehingga subjek --> objek, objek --> subjek

Structure passive voice in present continous :

S + to be + being + V3 + (by agent) + O

by phrase tidak ditulis tidak apa apa.

Identifikasi kalimatnya :

Mawar is eating mujair fish with dabu-dabu.

Mawar = Subjek, maka dia akan menjadi by agent

is = to be

eating = V3 nya adalah "eaten"

mujair fish = Objek, maka akan menjadi subjek

with dabu-dabu = Adverb

Sehingga kita susun kalimatnya sesuai struktur :

Mujair fish is being eaten (by Mawar) with dabu-dabu.
